# 📸 EXIF-Машина (EXIF Location Viewer)

Мощный OSINT-инструмент на Python для автоматического извлечения скрытых GPS-метаданных (EXIF) из фотографий и визуализации точного места съемки на интерактивной карте. Скрипт находит скрытый цифровой паспорт снимка и показывает, где именно он был сделан.

## 🚀 Функционал
* Глубокое сканирование файлов изображений (.jpg, .jpeg) на наличие метаданных.
* Автоматическое извлечение и декодирование координат (Широта и Долгота).
* Генерация готовой интерактивной HTML-карты на базе стабильных серверов CartoDB (работает без блокировок).

## 📋 Требования к операционной системе
Программа работает на любой из следующих операционных систем:
* 📱 Android (с использованием приложения Termux)
* 🐧 Linux (системы на базе Debian, включая Kali Linux)
* 💻 Unix

## Как установить

* Откройте терминал и введите `<pip install pillow folium --break-system-packages>`
* Затем `<git clone https://github.com/b5ediifgf/exif_viewer.git>`
* `<cd exif_viewer>`
* `<python3 exif_scanner.py>`
